# YouTalk — блог о психологии (Учебный проект)

Адаптивная вёрстка двух страниц: главная (лента статей) и страница статьи.

## Файлы

- `index.html` — главная
- `article.html` — статья
- `css/style.css` — стили
- `images/`, `icons/` — медиа

## Запуск

Откройте `index.html` в браузере.

## Технологии

HTML5, CSS3 (Grid, Flexbox, медиазапросы), БЭМ, доступность, адаптив.

## Адаптив

Десктоп → планшет → мобильный телефон (три брейкпойнта).
